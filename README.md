# Computer Architecture Lab – Branch Prediction and Cache Replacement Study

## Overview

This project implements and evaluates branch prediction and cache replacement policies using the ChampSim simulator.

The work includes:

### Task 1 – Branch Prediction
- Bimodal
- GShare
- Perceptron
- Hashed Perceptron
- Modified Bullseye Predictor
- Adaptive Bullseye Predictor

### Task 2 – Cache Replacement
- LLC Size Sensitivity Study
- LLC Associativity Study
- LRU
- SRRIP
- DRRIP
- SHIP
- MRU (Implemented)
- Random (Implemented)
- HC-RRIP (Implemented)
